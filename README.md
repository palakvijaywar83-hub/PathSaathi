# PathSaathi 📚 — Offline Study Buddy for Students

**Team:** NeuralNagpur
**Event:** Build with Gemma: Nagpur
**Direction:** Edge / On-Device

## Problem
Students in low-connectivity areas around Nagpur struggle to access AI-based tutoring because most tools need stable internet.

## Solution
PathSaathi is an offline-first AI study companion powered by Gemma. It runs a quantized Gemma 2B model that answers questions and generates practice questions, without needing an internet connection after setup.

## Why Gemma
Gemma's small, efficient model size (2B) is built to run on modest hardware, making true on-device offline deployment possible.

## Tech Stack
Google Gemma 2 (2B-it), Hugging Face Transformers, Python, Google Colab
